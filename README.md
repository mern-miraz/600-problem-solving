# 💻 JavaScript Final Projects

A collection of practical JavaScript projects built to strengthen core programming concepts, DOM manipulation skills, problem-solving ability, and real-world application development experience.

This repository contains multiple beginner-to-intermediate level JavaScript projects created as part of continuous learning and hands-on practice.

---

## 📖 About The Repository

The purpose of this repository is to apply JavaScript concepts through real projects instead of only theoretical exercises. Each project focuses on improving logical thinking, UI interaction handling, and overall front-end development skills.

These projects are designed to help developers gain practical experience with:

- JavaScript Fundamentals
- DOM Manipulation
- Event Handling
- User Interaction
- Dynamic UI Updates
- Problem Solving
- Project Structuring

---

## 🚀 Technologies Used

- JavaScript (ES6+)
- HTML5
- CSS3
- DOM API

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/mern-miraz/600-problem-solving.git
```

### Navigate to the Project Folder

```bash
cd 600-problem-solving/js-final-projects
```

### Run the Projects

Simply open the `index.html` file of any project in your browser.
